# 			1，运行级别

| 运行级别 |                           含义                            |
| :------: | :-------------------------------------------------------: |
|    0     |                           关机                            |
|    1     | 单用户模式，可以想象为windows的安全模式，主要用于系统修复 |
|    2     |              不完全的命令行模式，不含NFS服务              |
|    3     |            完全的命令行模式，就是标准字符界面             |
|    4     |                         系统保留                          |
|    5     |                         图形模式                          |
|    6     |                          重启动                           |

# 			2，运行级别命令

[root@penguin ~]# runlevel

\# 查看运行级别命令

[root@penguin ~]# init 运行级别

\# 改变运行级别命令

# 			3，系统默认运行级别

[root@penguin ~]# vim/etc/inittab

id:3:initdefault:

\# 系统开机后直接进入哪个运行级别
